## What I do

I break mobile apps for a living — finding the bugs before someone with worse intentions does.

- **Static analysis** of iOS (`.ipa`) and Android (`.apk`) binaries — decompilation, secret
  hunting, third-party SDK inventory, dependency / manifest review.
- **Runtime analysis** on jailbroken iOS and rooted Android — Frida instrumentation, method
  hooking, traffic interception, certificate-pinning bypasses, keychain / keystore extraction.
- **Vulnerability assessment** aligned with **OWASP MASVS / MASTG** — platform interaction,
  data storage, network, cryptography, code quality, and resilience.
- **Reporting** — structured findings with severity calibration, evidence standards, and
  remediation guidance.

## Toolbox

- **Static:** jadx · apktool · MobSF · class-dump · Hopper / Ghidra
- **Dynamic:** Frida · Objection · LLDB · drozer · logcat
- **Network:** Burp Suite · mitmproxy
- **Lab:** Corellium · physical jailbroken / rooted devices
